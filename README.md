# Hangman Game
A hangman game that allows users to select a letter to figure out a hidden word in a set amount of chances built in üç¶ vanilla JavaScript, HTML, CSS, and SVG.
- Display hangman pole and figure using SVG
- Generate a random word
- Display word in UI with correct letters
- Display wrong letters
- Show notification when select a letter twice
- Show popup on win or lose
- Play again button to reset game

2. Right click on `index.html` and open directly in the browser _or_ with VSCode's Live Sever extension.
